## 🚀 使用指南

### 快速开始

#### 1. 环境准备
```bash
# 确保安装了必要的工具
node --version    # >= 18.0.0
pnpm --version    # >= 8.0.0
```

#### 2. 项目安装
```bash
# 安装所有依赖
pnpm install

# 启动开发服务器
pnpm dev          # 同时启动 web 和 api

# 或分别启动
pnpm dev:web      # Web 应用 (端口 3000)
pnpm dev:api      # API 服务 (端口 3001)
```

#### 3. 环境配置
```bash
# 复制环境变量文件
cp .env.example .env

# 编辑配置文件
# - 设置数据库连接
# - 配置 AI 服务密钥
# - 设置其他环境变量
```

### 开发工作流

#### 1. 添加新功能
```bash
# 1. 创建功能分支
git checkout -b feature/new-feature

# 2. 开发功能
# - 在对应的 app 或 package 中开发
# - 使用共享类型和工具

# 3. 测试功能
pnpm test

# 4. 提交代码
pnpm lint:fix
git commit -m "feat: add new feature"
```

#### 2. 包间依赖
```bash
# 在 apps/web 中使用共享包
import { ApiClient } from '@repo/shared';
import { User, ChatThread } from '@repo/types';

# 在 apps/api 中使用共享类型
import { CreateThreadRequest } from '@repo/types';
```

#### 3. 构建和部署
```bash
# 构建所有应用
pnpm build

# 构建特定应用
pnpm build:web
pnpm build:api

# 生产环境启动
pnpm start
```
